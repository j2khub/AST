# 키움 API 자동매매 시스템

키움증권 RestAPI를 활용한 실시간 조건검색 기반 자동매매 시스템

---

## 🎯 프로젝트 목표

조건검색식에 편입된 종목을 실시간으로 감지하여 자동으로 매수 주문을 실행하는 시스템 구축

---
## 🔧 개발 시 주의사항

### 필드 매핑 주의
- `secretkey` (not `appsecret`)
- `token` (not `access_token`)
